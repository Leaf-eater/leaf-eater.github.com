##如何在基于hexo的博客中插入图片
在写FirstBlog中因为牵扯要插入以上图片嘛，开始的想法很简单file\\\D:\blog\picture\需要的图片 这样(也许是一时脑抽。。现在看来还真是不可理喻)。好吧，果然部署不上去，之后老老实实的google…吐槽完毕，下面是正文：
1.编辑你的配置文档_config.yml，找到并更改设置为post_asset_folder: true。这个设置的意思是设置一种模式：当你新建了一个名为 newBlog 的文章时，会自动创建一个名为 newBlog 的资源文件夹，这个文件夹用于存放这边博客所需的资源文件比如我们要插入的图片。
2.在hexo的目录下执行命令npm install https://github.com/CodeFalling/hexo-asset-image --save 。在输入前请确认你已经安装了npm。这里安装了一个github上的一个工具，用于方便的调用本地图片(github地址)。
3.接下来就是使用啦，这里可以使用markdown的正常语法调用图片![图片配字](newBlog/需要调用的图片.jpg)，这时候你就可以看到图片正常显示啦~。
